# GAMES-101

Environment: WSL2 Ubuntu 18.04 or 20.04

update apt
```
sudo apt update && sudo apt upgrade
```

install g++ gdb cmake
```
sudo apt install g++ gdb cmake
```

install opencv and eigen3
```
sudo apt install libopencv-dev libeigen3-dev
```
